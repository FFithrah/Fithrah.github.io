TEKAN SALAH SATU TOMBOL DI BAWAH UNTUK MEMPELAJARI WEBSITE INI

Menu Navigasi adalah sebuah program yang berfungsi untuk menghubungkan antara halaman website yang satu dengan halaman yang lain.

pada umumnya, codingan menu navigasi dibuat di bagian awal dari body dalam bentuk list

agar tampilan menu navigasi menjadi seperti gambar pertama, dapat diperindah di bagian css

css merupakan sebuah program untuk mengatur dan memperindah visual dari website yang kita buat

java script adalah program yang berfungsi untuk menampilkan data

pada website ini, script digunakan untuk menampilkan tampilan form untuk mengedit profile

codingan script dapat langsung ditulis di dalam data html tanpa harus ditulis di css

form adalah tempat responden dapat mengisikan sesuatu sesuai dengan keterangan pada website
